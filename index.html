Research

I'm interested in computer architecture, workload characterization, and efficient system design for AI. My work explores how deep learning workloads behave on modern hardware and how we can co-optimize architectures and runtime systems for performance and scalability. Lately, I've focused on analyzing AlphaFold3 and large language model serving pipelines to uncover inefficiencies in computation, memory access, and GPU utilization.
